About Me

Greetings! I'm Aiman, a game programmer and designer. I have been making games since I was 12. From making 2D flash games for multiple school events/competitions to building AAA level games for companies, I continuously challenge myself to solve new problems and create unforgettable experiences for players.
I am a Computer Science Engineering graduate from Cochin University of Science And Technology (CUSAT) with more than 1 year of experience in the gaming industry and over 5 years of experience with Unity3D game engine.
Here you can see some of my projects, and some of the personal-fun-things I've been working on. I have a strong perseverance for learning new things and taking on challenges, and I always like to put some extra time in polishing my creations, to make sure the creative vision is achieved in the best possible way.

Survival Shooter

A survival shooter game that can be played as First Person or Third Person (switchable at any point in the game). Created an advanced animation system for movement and shooting using Animation Rigging. Implemented fluid animation mechanics using blend trees. Used Inverse Kinematics for gun movement (gun sways on looking around and moving). Added all features that can be found in AAA games such as recoil effects, muzzle flashes, bullet tracers and decals. Used Cinemachine for Third Person Camera and custom scripts for First Person Camera. Implemented procedural shakes on camera, rig and weapon to achieve realistic effect when a weapon is fired. The game is currently under development and many features will be added soon.

Rewind

A full fledged high graphics game made in 1 week for the international Brackey's Game Jam. Since "Rewind" was the theme for the game jam, the mechanics is made such that the player can rewind time within the game, ie, all the NPCs, environment rewinds back in time according to the player's need to progress through all the levels. The game has recieved great response from the players, who were astounded to know that the game had been made in 1 week.

Corona GO

An augmented reality game developed during the first Corona virus lockdown. The title of the game is inspired from Pokemon GO. The player can play as a miniature doctor in their room jumping across table to floor to other objects in their environment shooting virus that spawns around it. The game uses computer vision to detect the objects in the camera view and convert them to colliders for the physics engine to calculate. This is a wave based game, thus viruses get spawned and the player has to pickup health orbs and new weapons and survive.

iDroid

A 2.5D platformer game for PC. A quick game made within 1 day using only unity's built in game objects (cube, cylinder, capsule, etc..) which takes in simple input (WASD or arrow keys) to control the character. Smooth game mechanics have been implemented, along with point system, checkpoint system and enemy objects. The player must go through the game in a single direction facing obstacles and traps each with it's own behaviour.

Quix

A prototype of a live quiz game for mobile. A question from a particular genre is shown along with it's corresponding options. The fastest player to answer within the time period wins the round. A random question is shown every round (no queston is repeated in the entire game session) with the options. Implemented complex data structures to handle the data. Used tweeing for the UI animations for faster and better performance. The UI design is made minimal and attractive for a good and long gameplay experience (Prototyped using Adobe XD).

Card Game

A prototype of a card game with 2 other opponents. The opponent AI behaves according to their cards and the cards dealt by others. Smooth game mechanics are implemented, ie, the card movement animation is done using tweeing, all cards in the player's hand are sorted (optimized sorting algorithms), cards are stacked using Unity's table feature. All UI elements are designed using Adobe Photoshop.

Asteroids

Inspired from the original Atari Asteroids game (1979), this game is made entirely using OpenGL (GLUT) and C++ without any game engine. Developed from scratch by drawing points to lines to triangles to different shapes, this game takes the arrow keys input for movement and spacebar to shoot. It is a level based survival shooter game and thus has infinite levels and a dedicated score system.
